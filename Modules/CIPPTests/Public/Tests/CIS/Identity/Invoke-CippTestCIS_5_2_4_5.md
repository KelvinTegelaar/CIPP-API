Administrator password resets are high-value events. Notifying all administrators when any admin resets their password provides peer visibility, so an unexpected reset of a privileged account is noticed quickly even if the targeted admin is unavailable.

**Remediation Action**

Microsoft Entra admin center > Entra ID > Password reset > Notifications > set **Notify all admins when other admins reset their password** to **Yes**.
