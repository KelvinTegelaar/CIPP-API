Email OTP delivers MFA codes to a guest's mailbox — if that mailbox is compromised, the second factor is too. CIS recommends disabling email OTP and forcing guests to create a Microsoft account.

**Remediation Action**

Microsoft Entra > Authentication methods > Email OTP — set to Disabled.
