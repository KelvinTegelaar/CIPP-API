"Stay signed in" extends sign-in lifetimes which is fine on managed devices but high-risk on shared / unmanaged devices. CIS recommends hiding the option globally and granting persistent sessions only to compliant devices via Conditional Access.

**Remediation Action**

Microsoft Entra admin centre > User experiences > Company branding > Default sign-in > Show option to remain signed-in: **No**.
