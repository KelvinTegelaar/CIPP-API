Notifying a user when their own password is reset gives the account owner an immediate signal if the reset was not initiated by them — an early warning of an account-takeover attempt.

**Remediation Action**

Microsoft Entra admin center > Entra ID > Password reset > Notifications > set **Notify users on password resets** to **Yes**.
