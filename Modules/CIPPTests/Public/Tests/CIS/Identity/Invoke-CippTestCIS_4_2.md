Personal device enrollment can give attackers a foothold via lost / stolen / unmanaged devices. CIS recommends blocking by default and requiring explicit allowlisting.

**Remediation Action**

Intune > Devices > Enrollment > Device platform restrictions > Default policy — set Personally owned to Block on every platform.
