# Standard Alignment Report

This report shows how well your tenant aligns with the standards templates that have been assigned to it. Each template defines a set of security and configuration policies that your managed service provider has chosen as the baseline for your environment.

A compliant standard means the setting is configured as expected. A non-compliant standard means the setting has drifted from the baseline or hasn't been applied yet. Items marked as license missing indicate that your current subscriptions don't include the features needed for that standard.
