# User MFA Report

This report covers all standard (non-admin) user accounts and their MFA enrollment status. While admin accounts carry the most risk, regular user accounts are still prime targets for phishing, credential stuffing, and social engineering attacks.

A single compromised user account can give an attacker a foothold to escalate privileges, exfiltrate data, or launch further attacks within your organization.
