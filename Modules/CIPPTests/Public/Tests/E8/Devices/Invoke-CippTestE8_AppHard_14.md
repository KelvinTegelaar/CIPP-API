Defender's advanced ransomware protection rule uses cloud-delivered ML to detect and block ransomware behaviour patterns even when the binary itself is unknown.

**Remediation Action**

1. Intune > Endpoint security > Attack surface reduction.
2. Set *Use advanced protection against ransomware* to **Block**.
3. Pair with Controlled Folder Access for additional anti-ransomware defence.
