Many drive-by downloads end with a JavaScript or VBScript stub that pulls down and runs a binary. This ASR rule terminates that hand-off.

**Remediation Action**

1. Intune > Endpoint security > Attack surface reduction.
2. Set *Block JavaScript or VBScript from launching downloaded executable content* to **Block**.
