ISM-1647 — privileges are reviewed at least annually. PIM eligibility assignments must therefore expire within 12 months so the next assignment is a deliberate decision. Permanent eligibility defeats the purpose of access reviews.

**Remediation Action**

1. Entra ID > PIM > Microsoft Entra roles > Assignments > Eligible.
2. For each assignment without expiry, set an end date ≤ 12 months from now.
3. Tighten role settings so future assignments cannot exceed 12 months.
