# Users Are Actively Using OneDrive and SharePoint for File Collaboration

Microsoft 365 Copilot delivers significant value when working with files — it can summarize documents, answer questions about file content, draft new documents based on existing ones, and surface relevant files in context. This capability is most impactful when users actively store and collaborate on files in OneDrive and SharePoint.

This test uses the Microsoft 365 Copilot Readiness report (30-day window) to check what percentage of licensed users worked on a document or file in OneDrive or SharePoint in the past 30 days. A threshold of 50% is used — users who never use cloud-stored files will not benefit from Copilot's document intelligence capabilities.
