# Users Are Actively Using Outlook Email

Microsoft 365 Copilot in Outlook helps users draft emails, summarize threads, and prepare for meetings by surfacing relevant context from their inbox. This value is only realized when users are active Outlook users — someone who never uses Outlook email will gain little from Copilot's email capabilities.

This test uses the Microsoft 365 Copilot Readiness report (30-day window) to check what percentage of licensed users sent or received email via Outlook in the past 30 days. A threshold of 50% is used — if fewer than half of licensed users are active in Outlook, email-centric Copilot features will have limited impact across the tenant.
