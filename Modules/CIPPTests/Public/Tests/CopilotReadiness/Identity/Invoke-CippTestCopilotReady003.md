# Users Have Activated M365 Desktop Apps

Microsoft 365 Copilot features in Word, Excel, PowerPoint, Outlook, and OneNote require the desktop versions of these apps — web-only usage does not provide access to Copilot within the applications. Users who have licenses for M365 desktop apps but have never activated them on a device will not be able to use Copilot in those apps.

This test cross-references the Office Activations report against licensed users to determine how many have activated M365 apps on at least one desktop device (Windows or Mac). A threshold of 70% is used — significantly below this suggests the tenant relies heavily on web/mobile usage and desktop Copilot features may have limited reach.

**Remediation action**
- [Deploy Microsoft 365 Apps with Microsoft Intune](https://learn.microsoft.com/en-us/deployoffice/deploy-microsoft-365-apps-intune)
- [Overview of the Microsoft 365 Apps admin center](https://learn.microsoft.com/en-us/deployoffice/admincenter/overview)
- [Microsoft 365 Apps activation report](https://learn.microsoft.com/en-us/microsoft-365/admin/activity-reports/microsoft365-apps-usage-ww)
