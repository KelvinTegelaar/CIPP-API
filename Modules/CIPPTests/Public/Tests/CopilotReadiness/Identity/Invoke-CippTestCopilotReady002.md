# Copilot Licenses Are Assigned to Users

Having purchased Microsoft 365 Copilot licenses is only the first step — licenses must also be assigned to individual users before they can access Copilot features. Unassigned seats represent unused investment and mean users cannot benefit from Copilot even if the tenant has purchased capacity.

This test checks that at least one Copilot license (e.g. `Microsoft_365_Copilot`, `COPILOT_STUDIO_IN_MICROSOFT_365`) is assigned to a user with `consumedUnits > 0`. It also surfaces the ratio of available seats to consumed seats so unused capacity is visible.

**Remediation action**
- [Assign Microsoft 365 Copilot licenses to users](https://learn.microsoft.com/en-us/microsoft-365/admin/manage/assign-licenses-to-users)
- [Manage Copilot licenses in the Microsoft 365 admin center](https://learn.microsoft.com/en-us/copilot/microsoft-365/microsoft-365-copilot-setup)
- [Set up Microsoft 365 Copilot](https://learn.microsoft.com/en-us/copilot/microsoft-365/microsoft-365-copilot-setup)
