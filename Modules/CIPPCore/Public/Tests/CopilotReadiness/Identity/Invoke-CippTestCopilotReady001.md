# Tenant Has M365 Copilot Prerequisite Licenses

Microsoft 365 Copilot requires users to have an eligible Microsoft 365 base license before a Copilot add-on license can be assigned. Without a qualifying prerequisite license, Copilot licenses cannot be assigned or activated, making this the foundational step for any Copilot deployment.

Eligible plans include Microsoft 365 E3, E5, Business Premium, F1, F3, and equivalent Office 365 and Exchange/SharePoint/OneDrive plans. This test checks that the tenant has at least one eligible SKU with available seats.

**Remediation action**
- [Microsoft 365 Copilot licensing requirements](https://learn.microsoft.com/en-us/copilot/microsoft-365/microsoft-365-copilot-licensing)
- [Compare Microsoft 365 plans](https://www.microsoft.com/en-us/microsoft-365/business/compare-all-plans)
- [Purchase Microsoft 365 Copilot](https://www.microsoft.com/en-us/microsoft-365/business/copilot-for-microsoft-365)
