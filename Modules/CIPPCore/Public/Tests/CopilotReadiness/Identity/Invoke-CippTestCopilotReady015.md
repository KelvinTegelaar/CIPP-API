# Microsoft 365 Copilot Usage Per User

Understanding which users are actively using Microsoft 365 Copilot helps MSPs and administrators measure the return on investment from Copilot licenses, identify power users who can serve as internal champions, and spot users who have licenses but have never engaged with Copilot and may need additional training or nudging.

This informational test retrieves per-user Copilot usage details from the Microsoft 365 usage reports API for the past 30 days, showing which apps each user has interacted with. No pass/fail is applied — the data is surfaced to give visibility into real-world Copilot adoption.
